# 📌 Project Overview

This project demonstrates the practical implementation of **Inferential Statistics** using Python. A healthcare dataset containing demographic information, lifestyle habits, and medical records is analyzed to derive meaningful statistical conclusions.

The project includes data preprocessing, descriptive statistics, hypothesis testing, confidence interval estimation, covariance, correlation analysis, ANOVA, Chi-Square Test, Independent T-Test, and data visualization.
